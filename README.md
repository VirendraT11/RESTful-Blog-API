## ğŸš€ Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/VirendraT11/RESTful-Blog-API.git
   cd RESTful-Blog-API
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up environment variables**:
